## 📂 Proyectos Open Source

Estoy preparando proyectos enfocados en resolver necesidades reales de desarrolladores, administradores de sistemas y empresas.

### 🔐 Odoo Module Security Scanner

Herramienta para analizar módulos de Odoo y detectar posibles problemas relacionados con:

* Permisos de acceso.
* Reglas de registro.
* Uso inseguro de `sudo()`.
* Archivos `ir.model.access.csv`.
* Consultas SQL.
* Configuración de seguridad.

### 🐘 PostgreSQL Health Check

Scripts para revisar el estado de servidores PostgreSQL:

* Conexiones activas.
* Consultas lentas.
* Bloqueos.
* Tamaño de bases de datos.
* Índices no utilizados.
* Estado de VACUUM.
* Consumo de almacenamiento.

### 🗄️ Oracle DBA Toolkit

Colección de consultas y herramientas para:

* Revisar tablespaces.
* Detectar usuarios bloqueados.
* Consultar sesiones activas.
* Analizar objetos inválidos.
* Revisar DBLinks.
* Automatizar respaldos y tareas administrativas.

### 🐧 Linux Server Toolkit

Scripts para automatizar:

* Respaldos.
* Monitoreo de CPU, memoria y disco.
* Revisión de servicios.
* Renovación de certificados SSL.
* Administración de logs.
* Seguridad básica de servidores.
